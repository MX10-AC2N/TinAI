**Modular Local AI Stack — 100 % offline — CPU optimized — fully customizable**

---

## ✨ Pourquoi TinAI ?

TinAI te permet de construire **ta propre stack IA locale** en choisissant exactement les outils dont tu as besoin.  
Un seul `make` → un beau menu interactif → tu sélectionnes tes services à la carte.

Tout est optionnel, léger et facile à étendre.

---

## 🚀 Démarrage rapide / Quick Start

```bash
git clone https://github.com/MX10-AC2N/TinAI.git && cd TinAI

cp .env.example .env          # personnalise si tu veux

make                          # ← menu interactif (recommandé)
